# comsol_auto_correct
本项目是一个面向 COMSOL Multiphysics 的智能自动化仿真系统，通过 OCR 文字识别、错误图标检测与 GUI 自动控制，实现 COMSOL 瞬态仿真过程中“报错识别→自动修改步长→重新计算”的全自动闭环。系统可自动检测“时步不收敛”“找不到一致初始值”等错误，并按预设规则迭代修改 `range(...)` 参数，无需人工干预即可持续试错，直到仿真稳定收敛或达到终止条件。项目基于 PaddleOCR、OpenCV 与 PyAutoGUI 构建，不依赖 COMSOL API，兼容性强，适用于两相流、虹吸、相场等易出现时步问题的复杂瞬态仿真场景，可实现长时间无人值守自动计算。
